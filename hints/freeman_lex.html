Ok, how exactly to fill in this dictionary?

I recommend you to use sets as values of dictionary to store only unique characters. Use dictionary get() method with default value equal empty set. Also use Set union() method to add character from the second string with specific index to the dictionary value set: 

dic = {}
for ind, char in enumerate(a):
    dic[char] = dic.get(char, set()).union(set(b[ind]))

So, for each unique character in the first string you add all the characters from the second string on the corresponding position.

So, how to check if the first string is isometric to the second?

Function should return True when to each unique character from the first string there is only one corresponding character from the second string. So, after updating the value set in dictionary make a check: if its length become more then one, then it is not 1-to-1 corresponding: 

if len(dic[char]) > 1:
    return False

On finishing iterating over the first string, if there were no False return, return True since the first string is isometric to the second.

So, how my final solution should look like?

Wrap the whole code into function expression. Here it is: 

def isometric_strings(a: str, b: str) -> bool:

    dic = {}
    for ind, char in enumerate(a):
        dic[char] = dic.get(char, set()).union(set(b[ind]))
        if len(dic[char]) > 1:
            return False

    return True

Use zip() function to create tuples of characters from two strings with the same index, keep unique ones with set(). The length if this set should be equal to the set of the first string. 

def isometric_strings(str1: str, str2: str) -> bool:

    return len(set(zip(str1, str2))) == len(set(str1))

String method str.maketrans() creates dictionary, where keys are characters from the first string and values - from the second. Then, using this dictionary, translate the first string. If the result is equal to the second string, return True. 

def isometric_strings(str1: str, str2: str) -> bool:

    trans = str.maketrans(str1, str2)
    
    return str1.translate(trans) == str2
