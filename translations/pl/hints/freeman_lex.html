Proszę złapać jakiś pomysł 💡! Generalnie chodzi o to, aby przejść przez pierwszy ciąg i dla każdego znaku pobrać i zapamiętać znak z drugiego ciągu na tej samej pozycji. Jeśli w pewnym momencie więcej niż jeden znak z drugiego ciągu odpowiada jednemu znakowi z pierwszego, to pierwszy ciąg nie jest izometryczny z drugim.

Od jakiego instrumentu powinienem zacząć?

Proszę zapoznać się z wbudowaną funkcją enumerate(). Podaje ona zarówno znak, jak i jego indeks w ciągu znaków. Proszę użyć pętli for do iteracji po tym obiekcie enumerate: 

for ind, char in enumerate(a):
    ...

Nie rozumiem, jak przechowywać odpowiadające sobie znaki w łańcuchach.

Potrzebuje Pan dictionary do przechowywania znaków z pierwszego ciągu jako kluczy i odpowiadających im znaków z drugiego ciągu jako wartości. Proszę utworzyć pusty słownik przed iteracją: 

dic = {}
for ind, char in enumerate(a):
    dic[char] = ...

Ok, jak dokładnie wypełnić ten słownik?

Zalecam użycie sets jako wartości słownika, aby przechowywać tylko unikalne znaki. Proszę użyć metody dictionary get() z wartością domyślną równą empty set. Proszę również użyć metody Set union(), aby dodać znak z drugiego ciągu o określonym indeksie do zestawu wartości słownika: 

dic = {}
for ind, char in enumerate(a):
    dic[char] = dic.get(char, set()).union(set(b[ind]))

W ten sposób dla każdego unikalnego znaku w pierwszym ciągu doda Pan wszystkie znaki z drugiego ciągu na odpowiedniej pozycji.

Jak więc sprawdzić, czy pierwszy ciąg jest izometryczny względem drugiego?

Funkcja powinna zwrócić True, jeśli dla każdego unikalnego znaku z pierwszego ciągu istnieje tylko jeden odpowiadający mu znak z drugiego ciągu. Proszę więc po zaktualizowaniu wartości w słowniku sprawdzić, czy jeśli jego długość jest większa niż jeden, to czy nie odpowiada on 1 do 1: 

if len(dic[char])> 1:
    return False

Po zakończeniu iteracji nad pierwszym ciągiem, jeśli nie było False powrót, powrót True ponieważ pierwszy ciąg jest izometryczny do drugiego.

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Proszę opakować cały kod w wyrażenie funkcyjne. Proszę bardzo: 

def isometric_strings(a: str, b: str) -> bool:

    dic = {}
    for ind, char in enumerate(a):
        dic[char] = dic.get(char, set()).union(set(b[ind]))
        if len(dic[char])> 1:
            return False

    return True

Jasne, proszę łapać!

Proszę użyć funkcji zip(), aby utworzyć krotki znaków z dwóch ciągów o tym samym indeksie, zachowując unikalne z set(). Długość tego zbioru powinna być równa długości pierwszego ciągu. 

def isometric_strings(str1: str, str2: str) -> bool:

    return len(set(zip(str1, str2))) == len(set(str1))

Metoda String str.maketrans() tworzy słownik, w którym kluczami są znaki z pierwszego łańcucha, a wartościami - z drugiego. Następnie za pomocą tego słownika proszę przetłumaczyć pierwszy ciąg znaków. Jeśli wynik jest równy drugiemu ciągowi, proszę zwrócić True. 

def isometric_strings(str1: str, str2: str) -> bool:

    trans = str.maketrans(str1, str2)
    
    return str1.translate(trans) == str2
